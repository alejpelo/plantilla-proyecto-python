# Librerías adicionas

## Uso

Para instalar una librería adicional al proyecto, se debe ejecutar, por ejemplo:

```sh
uv add numpy
```

Si la va a instalar solo para desarrollo (_dev_), ejecutar:

```sh
uv add nox --dev
```
